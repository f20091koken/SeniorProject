# クトゥアルト
ボードゲーム「Quarto」の用対戦AI  
モンテカルロ木探索を利用している。  
2018年2月頃に作成したQuarto用対戦AIを公開

もともとPython3で作成していたAIだが、  
Github Pagesで公開するためにクライアントで処理が完結するようにjavascriptに移植した。
